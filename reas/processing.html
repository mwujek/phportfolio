Processing

Processing is an open source programming language for people who want to create images, animations, and interactions. After Reas and Fry launched the program in 2001, tens of thousands of students, artists, designers, researchers, and hobbyists have used Processing for learning, prototyping, and production. Processing also won a Golden Nica from the Prix Ars Electronica in 2005. The project also received the 2005 Interactive Design prize from the Tokyo Type Director's Club.

This powerful application combines many subjects such as computer science, statistics, graphic design, and data visualization. The end result gives the user an animated composition as a mean for understanding and interpreting information. Initially developed to serve as a software sketchbook and to teach fundamentals of computer programming within a visual context, Processing also has evolved into a tool for generating finished professional work.
